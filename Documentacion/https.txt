Necesitamos incluir un certificado SSL/TLS pj, Lets encrypt
https://letsencrypt.org/getting-started/

Lets encrypt permite gestionar un cliente ACME ( Automatic Certificate Management Environment ) para 
operar con HTTPS

Podemos ver los comandos a utilizar dependiendo del servicio y sistema que usemos en:

https://certbot.eff.org/instructions

Para apache-LINUX(snap):

(Tener preparado un servidor con acceso ssh, online y con el puerto 80 abierto. Ademas, tener un usuario con acceso a sudo)

Instalar certbot

sudo apt-get install snapd
sudo apt-get remove certbot

sudo snap install --classic certbot
sudo ln -s /snap/bin/certbot /usr/bin/certbot

Tenemos dos opciones podemos instalar de forma automatica Certbot y permitir que haga cambios
en nuestra configuracion de Apache o bien instalarlo pero hacer los cambios a mano.

Automatico
sudo certbot --apache

Manual
sudo certbot certonly --apache
